# アーカイブ済みデザイン

このディレクトリには、現行UIの正本ではない過去資産を保管する。

- `initial-concepts/`: 実装前の SVG ベース初期構想
- `legacy-screens/`: 現在は正規画面ではない旧スクリーンショット
- `ToBe/`: 過去タスクで参照していた ToBe 原案
- `screen-updates/`: 現行スクリーンショット更新時に退避した旧 PNG 一式

`screen-updates/` には `YYYY-MM-DD/` 単位のフォルダを作り、その更新で置き換えた PNG をまとめて保管する。

現行の画面イメージを参照したい場合は、1つ上の [docs/designs/README.md](../README.md) を見る。
