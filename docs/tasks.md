# 実装タスク

このファイルは実装作業のタスクリストです。

仕様の正本は `docs/specs/` 配下です。タスク実行中に仕様変更が発生した場合は、関連するSSOT文書と `.codex/config.toml` を同じ作業内で更新してください。

## 0. プロジェクト基盤

- [ ] ルートに `backend/` と `frontend/` を作成する。
- [ ] `backend/app/domain`, `backend/app/application`, `backend/app/infrastructure`, `backend/app/presentation` を作成する。
- [ ] FastAPIプロジェクトを初期化する。
- [ ] Next.js App Routerプロジェクトを初期化する。
- [ ] MySQL 8.4用のローカル開発環境を用意する。
- [ ] Python/Nodeの依存管理方法を決めて設定する。
- [ ] 環境変数サンプルを作成する。
- [ ] ローカル起動手順をREADMEまたは開発ドキュメントに追記する。
