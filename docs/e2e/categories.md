# カテゴリ管理E2Eシナリオ

## 対応テストファイル

- `frontend/e2e/categories.spec.ts`

## 目的

カテゴリ管理で初期カテゴリ表示、カテゴリ追加、編集、無効化、有効化、削除、分類ルール説明、未分類確認リンクを確認する。

## 前提データ

- 認証済みstorage stateを使う。
- サンプルカテゴリとして `食費` と `未分類` が含まれる。
- 追加するカテゴリ名は `E2Eカテゴリ` とする。

## 操作手順

- `/categories` を開く。
- `食費` と色スウォッチを確認する。
- カテゴリを追加する。
- 追加したカテゴリを編集する。
- 編集したカテゴリを無効化し、有効化する。
- 編集したカテゴリを削除する。
- 分類ルール説明ダイアログを開閉する。
- 未分類の確認リンクをクリックする。

## 期待結果

- `食費` が表示され、指定色のスウォッチが表示される。
- 追加したカテゴリが一覧に表示される。
- 編集後のカテゴリ名と説明が表示される。
- 無効化後は `無効`、有効化後は `有効` が表示される。
- 削除後、編集済みカテゴリは表示されない。
- 分類ルール説明ダイアログを開閉できる。
- 未分類確認リンクで `/transactions?category_id=...&period=all` へ遷移し、日付未指定、未分類カテゴリ絞り込みの状態で未分類明細が表示される。

## 安定化メモ

- 作成したE2Eカテゴリは同じテスト内で削除する。
- 削除確認ダイアログはPlaywrightのdialog handlerで受ける。
- 未分類リンクのURLはカテゴリIDを固定せず、パターンで確認する。
