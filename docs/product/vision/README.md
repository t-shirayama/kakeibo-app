# プロダクトビジョン

## 目的

kakeibo-app は、楽天カード明細PDFから取引明細を取り込み、日々の支出・収入をカテゴリ別に管理するためのWebアプリです。明細の確認・編集、カテゴリ管理、店名キーワードによる分類ルール、月別・週別・年別の集計、Excel出力までを一つの画面群で扱います。

## 主な課題

- PDF明細から家計簿へ転記する作業に手間がかかる。
- 取り込んだ明細のカテゴリ付けや修正が継続運用の負担になりやすい。
- 収入、支出、予算を別々に管理すると、月ごとの家計状況を振り返りにくい。
- 明細、カテゴリ、分類ルール、アップロード履歴が分散すると、取込結果の確認や修正に時間がかかる。

## 解決すること

- 楽天カード明細PDFをアップロードし、明細として取り込めるようにする。
- 明細の確認、追加、編集、カテゴリ変更を同じ家計簿データ上で扱えるようにする。
- 店名キーワードによる分類ルールで、PDF取込時のカテゴリ設定を補助する。
- 収入設定と月別変更を管理し、支出だけでなく収入も月次の振り返りに含める。
- 予算と実績を比較し、月別・週別・年別の集計で家計の変化を追いやすくする。
- Excel出力やレポートにより、画面外でも振り返りや共有がしやすい状態にする。

## 大事にすること

- 取込結果をユーザーが確認・修正しやすいこと。
- カテゴリや分類ルールを育てるほど、日々の入力負担が軽くなること。
- 収入、支出、予算、レポートが同じ前提でつながり、月次の判断に使いやすいこと。
- 失敗したPDF取込や削除操作など、家計データに関わる重要な操作の状態が追えること。
- 仕様、画面要件、用語が整理され、継続的に機能を追加しても迷わず保守できること。
