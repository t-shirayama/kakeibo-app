# カテゴリ管理要件

## 対応デザイン

- `docs/designs/categories.png`
- `docs/designs/categories.svg`

## 目的

明細分類に使うカテゴリを管理できるようにする。

## 表示項目

- カテゴリ名
- 色
- 説明
- 編集操作
- 削除操作

## 操作

- カテゴリを追加できる。
- カテゴリを編集できる。
- カテゴリを削除できる。
- カテゴリ色を設定できる。

## 業務ルール

- カテゴリ名は必須とする。
- 同一ユーザー内でカテゴリ名の重複を許可しない。
- 色は画面上で識別できる値を設定する。
- 既存明細で使用中のカテゴリは物理削除しない。
- 使用中カテゴリを削除する場合は、無効化または未分類への付け替えを検討する。
- 初期カテゴリとして、食費、交通費、日用品、交際費、趣味・娯楽、通信費、その他を用意する。

## 例外・空状態

- カテゴリがない場合は、追加を促す空状態を表示する。
- 重複名で保存しようとした場合は、入力エラーにする。
- 削除できないカテゴリは理由を表示する。

## 確認事項

- 使用中カテゴリの削除時は「無効化」と「未分類へ付け替え」のどちらを採用するか。
- 初期カテゴリはユーザーごとに複製するか、システム共通カテゴリとして扱うか。
