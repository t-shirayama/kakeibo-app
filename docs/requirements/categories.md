# カテゴリ管理要件

## 対応デザイン

- `docs/designs/categories.png`

## 目的

明細分類に使うカテゴリを管理できるようにする。

## 表示項目

- カテゴリ名
- 色
- 説明
- 編集操作
- 削除操作

## 操作

- カテゴリを追加できる。
- カテゴリを編集できる。
- カテゴリの有効/無効を切り替えられる。
- カテゴリを削除できる。
- カテゴリ色を設定できる。
- 店名キーワードの自動分類ルール画面へ遷移できる。
- 未分類カテゴリの明細を明細一覧で確認できる。

## 業務ルール

- カテゴリ名は必須とする。
- 同一ユーザー内でカテゴリ名の重複を許可しない。
- 色は画面上で識別できる値を設定する。
- 既存明細で使用中のカテゴリは物理削除しない。
- 使用中カテゴリは物理削除せず、既存明細のカテゴリ参照を維持したまま論理削除または無効化する。
- 無効化または論理削除されたカテゴリに紐づく明細は、明細一覧では「未分類」として確認できる。
- 初期カテゴリとして、食費、交通費、日用品、交際費、趣味・娯楽、通信費、その他を用意する。
- 初期カテゴリはユーザー作成時にユーザーごとに複製する。

## 例外・空状態

- カテゴリがない場合は、追加を促す空状態を表示する。
- 重複名で保存しようとした場合は、入力エラーにする。
- 削除できないカテゴリは理由を表示する。
