# 設定要件

## 対応デザイン

- `docs/designs/settings.png`
- `docs/designs/settings.svg`

## 目的

ユーザーが家計簿アプリの表示、データ、基本設定を変更できるようにする。

## 表示項目

- 通貨
- タイムゾーン
- 1ページあたりの件数
- 日付の表示形式
- データエクスポート
- 全データ削除
- ダークモード

## 操作

- 通貨を設定できる。
- タイムゾーンを設定できる。
- 一覧の表示件数を設定できる。
- 日付表示形式を設定できる。
- データをエクスポートできる。
- すべてのデータを削除できる。
- ダークモードを切り替えられる。

## 業務ルール

- 初期通貨はJPYとする。
- 初期タイムゾーンはAsia/Tokyoとする。
- 日付表示形式の初期値は`YYYY/MM/DD`とする。
- 全データ削除は確認後に実行する。
- 全データ削除の対象はログインユーザー本人のデータに限定する。
- 設定変更後は次回表示から反映する。

## 例外・空状態

- 設定保存に失敗した場合は、変更前の状態を維持する。
- 全データ削除に失敗した場合は、削除結果が不明にならないようにエラーを表示する。
- エクスポート対象データがない場合は、空データとして扱うか、実行不可にする。

## 確認事項

- 通貨はJPY固定でよいか、複数通貨対応が必要か。
- ダークモードは初期リリースで実装するか。
- 全データ削除時にパスワード再入力などの追加確認を必要とするか。
