# レポート（グラフ）要件

## 対応デザイン

- `docs/designs/reports.png`
- `docs/designs/reports.svg`

## 目的

支出の傾向をグラフで把握し、カテゴリ別・月別の分析をできるようにする。
現在は独立画面としては提供せず、ダッシュボード画面へ統合する。

## 画面導線

- `/dashboard` をレポート統合画面の正規ルートとする。
- `/reports` は既存ブックマーク互換のため `/dashboard` へリダイレクトする。

## 操作

- 対象年月を選択できる。
- レポートをExcel（`.xlsx`）でエクスポートできる。
- 詳細な期間確認は明細一覧画面へ遷移して行う。

## 業務ルール

- レポートは支出明細を対象にする。
- 対象期間内の明細をカテゴリ別に集計する。
- 月別収入・支出推移は月単位で収入合計と支出合計を集計する。
- レポートのエクスポート形式はExcel（`.xlsx`）とする。
- Excel出力には明細一覧、カテゴリ集計、月別集計を含める。
- レポート画面のExcelエクスポート導線はダッシュボード画面へ統合する。

## 例外・空状態

- 対象期間に明細がない場合は、集計値を0として空状態を表示する。
- グラフ描画に必要なデータがない場合は、グラフ領域に空状態を表示する。
- エクスポートに失敗した場合は、再実行できるエラー表示にする。
