# 分類ルール要件

## 目的

PDF取込時に、店名キーワードからカテゴリを自動設定できるようにする。

## 表示項目

- 店名キーワード
- 分類先カテゴリ
- 有効状態

## 操作

- 分類ルールを追加できる。
- 分類ルールを編集できる。
- 分類ルールの有効/無効を切り替えられる。
- 分類ルールを削除できる。
- カテゴリ管理画面から分類ルール画面へ移動できる。

## 業務ルール

- 店名キーワードは必須とし、trim後に保存する。
- 同一ユーザー内で論理未削除の同一キーワード重複を許可しない。
- 分類先カテゴリは有効な自分のカテゴリのみ指定できる。
- PDF取込時は、店名とキーワードをtrim + 小文字化して部分一致で判定する。
- 複数の有効ルールに一致した場合は、キーワードが長いルールを優先し、同じ長さなら新しいルールを優先する。
- 無効ルール、削除済みルール、無効カテゴリに紐づくルールは自動分類に使わない。

## 画面状態

- 分類ルールがない場合は空状態を表示する。
- 有効カテゴリがない場合は、先にカテゴリ追加が必要なことを表示する。
- APIエラー時は共通エラー表示を出す。
