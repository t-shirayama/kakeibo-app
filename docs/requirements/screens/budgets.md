# 予算管理要件

## 目的

カテゴリごとの月次予算を設定し、対象月の予実を一つの画面で切り替えて確認できるようにする。

## 表示項目

- 予算設定タブ
- 予実確認タブ
- カテゴリ名
- 色
- 有効/無効状態
- 月次予算
- 対象月の予算合計
- 対象月の支出実績
- 残額または超過額
- 進捗率
- カテゴリ別予実

## 操作

- 予算設定タブと予実確認タブを切り替えられる。
- 予算設定タブでカテゴリごとの月次予算を設定、変更、解除できる。
- 予実確認タブで表示月を `type="month"` 入力と前月/翌月ボタンで切り替えられる。
- 表示月はURLクエリへ同期し、再読み込み後も同じ月を復元できる。

## 業務ルール

- 予算設定ではカテゴリ名、色、説明の編集は行わず、月次予算だけを扱う。
- 月次予算は未設定または0以上の整数JPYとする。
- 予算合計、支出実績、残額または超過額、進捗率は対象月のデータから算出する。
- 進捗率は `支出実績 / 予算 * 100` で算出し、予算未設定または0円の場合は0%として扱う。
- カテゴリ別予実は月次予算が設定されたカテゴリだけを表示する。
- 予算を超過したカテゴリや全体進捗は、超過額が分かるように表示する。
- カテゴリ名、色、説明の変更や有効/無効の切り替えはカテゴリ管理画面で行う。

## 例外・空状態

- カテゴリがない場合は、カテゴリ追加を促す空状態を表示する。
- 月次予算が未設定の場合は、予実確認タブで設定を促す空状態を表示する。
- 集計取得または予算保存に失敗した場合は、再実行可能なエラー表示にする。
