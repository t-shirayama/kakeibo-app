# テストドキュメント

このディレクトリはテスト方針とシナリオ文書の入口です。

## 構成

- [単体テスト](unit/README.md): ドメイン、ユースケース、UI部品など小さな単位の確認方針。
- [結合テスト](integration/README.md): Backend Integration Test と Frontend Integration Test の確認方針。
- [E2Eテスト](e2e/README.md): 実ブラウザ、実バックエンド、MySQLを含む代表導線の確認方針。

テスト方針全体の正本は [開発・運用ワークフロー](../specs/development-workflow/README.md) です。
