# 予算管理E2Eシナリオ

## 対応テストファイル

- `frontend/e2e/budgets.spec.ts`

## 目的

予算管理画面で、予算設定タブと予実確認タブを切り替えられ、対象月の予算進捗を確認できることを検証する。

## 前提データ

- 認証済みstorage stateを使う。
- サンプルカテゴリとして `食費` が含まれる。
- サンプルデータには対象月のカテゴリ別予算進捗が入っている。

## 操作手順

- `/budgets` を開く。
- 予算設定タブが初期表示されることを確認する。
- `食費` の予算設定が表示されることを確認する。
- 予実確認タブへ切り替える。
- 表示月を前月へ切り替える。
- ページを再読み込みする。

## 期待結果

- 予算設定タブと予実確認タブを切り替えられる。
- 予算設定タブでカテゴリごとの月次予算が表示される。
- 予実確認タブで全体進捗とカテゴリ別予算進捗が表示される。
- 表示月は `YYYY-MM` 形式で表示され、前月・翌月ボタンで切り替えられる。
- 再読み込み後も選択した表示月が維持される。

## 安定化メモ

- 表示月の待ち合わせは `frontend/e2e/helpers/date.ts` の helper を使う。
- 金額の厳密値はAPI / Integration Testへ寄せ、E2Eでは代表データの表示を確認する。
